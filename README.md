TO USE:

Download IRC.zip

Unzip (WARNING: the paths will be LONG. I recommend enabling long paths and using 7zip)

Either use as a regular collection of markdown files, or install Obsidian and designate the folder as a vault to take full advantage of YAML and Linking

Enjoy the tax code
